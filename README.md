Music Store Data Analysis (SQL)

A comprehensive SQL-based analysis of a digital music store's database .
This project explores key business metrics, customer purchasing patterns, artist performance, and regional sales distribution using progressively complex database queries.

Project Overview-:

The project is structured into three analytical levels:
1. **Basic Analysis:** Employee seniority, invoice distribution, top revenue-generating cities, and primary customer spending.
2. **Intermediate Analysis:** Customer preferences by genre, rock music artist analysis, and track duration metrics.
3. **Advanced Analysis:** Customer spending per top artist, regional genre popularity using Window Functions, and country-level top spender determination.
